**Overview:**

**PROJECT \#2 – ANALYZE**

*Client Prompt:*

As part of an internal effort to streamline calculations, your structural engineering firm has requested that you develop a Python tool to perform standard statics calculations. Your supervisor has provided you with a sample implementation. You are also provided with solutions to the statics problems to be included in a pilot product to validate your code.

Your firm specifically asks that the tool include the following items in its functionality:

1.  The ability to perform unit conversion and analysis.
2.  The ability to handle distributed and point loads acting on static members.
3.  The flexibility to provide consistent calculations in response to variable user inputs including load magnitude, location, and angle of application.

Your firm asks you to use your coding skills in Python over the next 3 weeks to complete the pilot tool development and provide a summary of the results to your professional engineer (PE) supervisor so your PE can discuss next steps with the full team of firm principals. The summary of the deliverables requested by your PE and the deadline are listed below.

*Deliverables:*

-   A Python code file that documents your code used to generate your solution.
-   A well-written scope of work related to the firm prompt that defines the tasks undertaken by your group to complete the work.
-   A GANTT chart outlining the project tasks for the completion and testing of both the pilot and full tool.
-   An annotated code document that explains each line of code in detail ([https://uofnelincoln-my.sharepoint.com/:w:/g/personal/klane11_unl_edu/EWQY3dadECBAppqDxlIi2UwByWjgxcJOOCsRPVudJPMq6Q?e=RDILxCLinks to an external site.](https://uofnelincoln-my.sharepoint.com/:w:/g/personal/klane11_unl_edu/EWQY3dadECBAppqDxlIi2UwByWjgxcJOOCsRPVudJPMq6Q?e=RDILxC) )
-   A written document that summarizes the findings contained in the code that can be handed to your firm for internal discussions.

*Skills Students will learn:*

-   Translate “client” desires and objectives into an itemized project task list
-   Sequence a set of tasks necessary to complete an engineering project
-   Recognize what data analysis method has been used in an engineering design project
-   Interpret Python code produced by fellow students, the instructor, and external resources on the internet
-   Carry out integration and differentiation operations in Python
-   Produce answers to statics problems using Python code
-   Calculate key concepts related to statics problem
