*Project Goals:*

The goal of this project is to create a tool that can perform standard statics calculations. This tool is to be created for the structural engineering firm. The tool will perform unit conversions and analysis, handle distributed and point loads acting on static members, and be able to accept variable inputs by the user and provide consistent outputs including load magnitude, location, and angle of application. A written summary is to be provided that summarizes the tool and its use.
